FLICKERSCAPE

About:

FLICKERSCAPE is an interactive multimedia artwork based in the web browser. It was created by Christopher Lock in 2020 and was written in HTML, CSS, and JavaScript. It uses the Web Audio API and the HTML audio tag for all of the sounds and CSS animations and JavaScript for the visuals. FLICKERSCAPE is currently not mobile freindly and for optimal experience headphones are recomended. Search for the squares that turn red, click them, and new sounds will be added to the texture.
